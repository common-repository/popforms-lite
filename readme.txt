== Installation ==

**The easy way.**

Go to your WordPress Dashboard. Navigate to Plugins > Add New and then search for "PopForms Lite". Click on Install and then Activate the Plugin.

That's it, PopForms Lite Plugin is now activated on your site!

**The hard way..**

Download "PopForms Lite WordPress Plugin"  and then extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
